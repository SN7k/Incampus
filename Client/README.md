# InCampus

A modern social networking platform tailored for academic communities - connecting students and faculty in a secure, focused environment.

![InCampus](https://via.placeholder.com/800x400?text=InCampus+Screenshot)

## ✨ Features

### 🔐 Authentication
- Email registration with OTP verification
- University ID integration
- Secure login system

### 👥 Social Features
- **Friends:** Add, manage requests, receive suggestions based on academic info
- **Posts:** Create, like, comment, and share content with images
- **Profiles:** Customize info, view galleries, track engagement
- **Notifications:** Real-time alerts for social interactions

### 🎨 Experience
- Responsive design (mobile & desktop)
- Dark/light theme toggle
- Intuitive search functionality

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Open http://localhost:5173
```

## 🛠️ Tech Stack

- **Frontend:** React with TypeScript
- **State Management:** Context API
- **Styling:** Tailwind CSS
- **Data Persistence:** localStorage (demo)
- **Build Tool:** Vite

## 📁 Project Structure

```
src/
 ├── components/        # UI components organized by feature
 ├── contexts/          # React Context providers
 ├── pages/             # Main application pages
 ├── types/             # TypeScript definitions
 └── data/              # Mock data (development only)
```
